# node-chat-one-to-one
Node socket.io based one to one chat.


## Features
 1. One to one chat 
 2. Chat Rooms
 3. Individual login using passport.
 4. Bootstrap 4 UI
 5. Seperate UI, so you can skin as you want.
 6. Developed with Express.js, EJS, Socket.io
